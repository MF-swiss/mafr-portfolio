Portfolio – Marco Fritsche


Dieses Repository enthält das persönliche Portfolio von Marco Fritsche. Die Website präsentiert Projekte, Fähigkeiten, berufliche Stationen und Kontaktmöglichkeiten in einem modernen, responsiven Design. Der Fokus liegt auf klarer Struktur, hochwertiger UI‑Gestaltung und einer professionellen Darstellung für Bewerbungen und Kundenprojekte.

🌐 Inhalte
- Startseite mit persönlicher Vorstellung
- Über mich mit beruflichem Hintergrund
- Projekte mit detaillierten Beschreibungen
- Skills mit technischen und methodischen Kompetenzen
- Downloads (z. B. Lebenslauf)
- Kontaktseite
Alle Seiten sind öffentlich zugänglich und ohne Login erreichbar


📁 Projektstruktu
portfolio/
├── index.html            # Startseite
├── home.html             # Alternative Startseite (falls genutzt)
├── projekte.html         # Projekte
├── skills.html           # Skills
├── downloads.html        # Downloads
├── kontakt.html          # Kontakt
├── ueber-mich.html       # Über mich
├── images/               # Bilder und Assets
├── style.css             # Globales Stylesheet
└── README.md             # Dokumentation

🎨 Design & Technik
- HTML5 für Struktur
- CSS3 für Layout, Animationen und Dark‑Mode
- Responsives Design für Desktop und Mobile
- Dark‑Mode mit LocalStorage, damit die Einstellung beim Reload erhalten bleibt
- Keine externen Frameworks → schnelle Ladezeiten, volle Kontrolle


🚀 Deployment
Das Portfolio kann problemlos auf GitHub Pages, Netlify oder jedem statischen Webserver gehostet werden.

📄 Lizenz
Dieses Projekt ist frei für persönliche und berufliche Nutzung.
